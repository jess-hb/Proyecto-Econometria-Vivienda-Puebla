# Análisis Econométrico: Mercado Inmobiliario en Puebla 🐺📈
**Equipo:** Lobo Métricos - BUAP

Este repositorio contiene el análisis de regresión lineal múltiple para determinar los factores que influyen en el precio de las viviendas en Puebla.

### Contenido:
- **Limpieza de Datos:** Filtro de 128k registros a 795 observaciones válidas.
- **Variables:** Superficie, Cuartos, Pisos y Distancia al Zócalo (Haversine).
- **Resultados:** R-cuadrado de 0.49 y validación de supuestos (Breusch-Pagan, VIF).

*Nota: Proyecto desarrollado para la materia de Econometría I.(2026 - Primavera)*
